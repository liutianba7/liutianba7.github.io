    📚 学习

    🌤️ 上午

    1、上午先复习了 asyncio 库，总算是稍微熟悉了这个库，比如，什么是事件循环（调度器）、事件循环启动需要一个主协程（调度中心）、协程（可以被中断的函数）、task（能够被调度的单元）、同时也掌握了 gather、wait、create_task 等用法，还有，最为重要的就是 await 后面可以直接跟协程对象，也可以根 task，但是本质是不同的，跟协程对象，代码是同步的，想要并发，必须把这个 coroutine 对象包装为 task 对象！

    import asyncio  
from datetime import datetime, date, time, timedelta  
import time  

async def task(name: str, delay: int) -> str:  
    print(f'{name} is running')  
    await asyncio.sleep(delay)  
    print(f'{name} is done')  

    return name  

async def main():  
    # 拿到当前时间  
    now = time.time()  

    # 1. 串行执行  
    await task('task1', 1)  
    await task('task2', 2)    # await task('task3', 3)    

    # 2. 并发执行  
    await asyncio.gather(  
        asyncio.create_task(task('task1', 1)),  
        asyncio.create_task(task('task2', 2)),  
        asyncio.create_task(task('task3', 3)),  
    )  
    # 3. wait  
    done, pending = await asyncio.wait(  
        [  
            asyncio.create_task(task('task1', 1)),  
            asyncio.create_task(task('task2', 2)),  
            asyncio.create_task(task('task3', 3)),  
        ],  
    )


if __name__ == '__main__':  
    asyncio.run(main())

    2、复习了 fastapi 的一些基本知识（这次算是搭建起来了一个基本的框架），从宏观来讲，fastapi 就是一个高性能、轻量、异步 web 框架，然后最基本的知识就是路由（路径 + 操作函数 + http 方法）的构成，接收和校验各种请求参数（query、path、body）、向客户端返回各种各样的响应（Json、PlainText、Html、Redirect）等，模块化路由（APIROUTER），异常处理（用 raise，对于自定义异常，可以注册一个异常处理器），至于其他内容，比如依赖注入（本质就是一一个函数，只不过这个函数的返回值可能会被很多地方用），中间件（拦截请求），没有复习，之后用到再说吧。
