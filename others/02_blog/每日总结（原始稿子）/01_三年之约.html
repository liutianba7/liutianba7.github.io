    01 三年之约

    在 2026.06.03，我也是属于再一次觉醒了，就像是大一下的某一天，决定好好学习，争取推免名额一样，我的决心从未如此强烈过。

    在昨天晚上的沉思之后，我大致从五个方面做了规划，这是一个长期规划，我称它为“三年之约”，我给现在的自己打分为 10 / 100（从家庭，外貌，身高，学历多个维度），这个计划的目的就是在不🍄的前提下，将自己提升到 60 / 100，毕业之后，赚了钱再去手术，将自己从 60 提升到 80 - 85 这个区间，对，大体就是这个规划。 对于学习方面的规划，就是随着不同时间，去动态调整，其他方面的，就是坚持坚持坚持！！！

    当然，虽然规划了多方面，但主线一定是： 技术成长 → 大厂实习 → 秋招竞争力 → 高质量职业起点。

    学习方面

    首先是学习方面，这个之前也确实一直在做，学习是为了拿到研究生学历，学习就业技术，未来找到一份比较赚钱的工作，在之前的一段时间，我主要学习了 java 后端、一些中间件如：redis, rmq, es 以及 python 后端和 agent 相关知识，同时八股能力和算法题能力也有了比较大的提升，在此基础上，也有了两段小厂实习，第二版简历也已经做好，如下：

    📎简历2.pdf

    接下来学习上的规划大方向来说就是巩固基础，后端和 ai 应用是保底，是基石，在开学前的这三个月，主要学习深度学习、nlp、模型后训练相关知识，同时同步的投递后端和 agent 岗位，早日找到第一份大厂实习。同时，最为重要的一点，就是八股和算法尽量不要占用正常时间，所以我打算取消午休，每天在原本的午休时间去复习八股 + 刷 3-5道算法题，这一策略是因为我现在每天上午除了八股和算法题，什么都不去做，属于是安慰自己，欺骗自己了！具体的学习路线和规划放在下面的“近期学习规划与记录”。

    健身和饮食

    学习可以是最重要的，最核心的，但它一定不是全部，学习只是让我有赚钱的能力，但是有一具健康的身体，是更加宝贵的财富，健身的主要目的是塑造自己的身材，降体脂，长肌肉。饮食则是辅助健身，同时也是避免去吃那些油腻的、甜的、不好的食物，等价于护肤了。具体的健身 + 饮食规划放在下面的 "近期健身和饮食规划" 中。

    自媒体与直播

    在这个时代，流量为王，你可以靠颜值、靠技术、靠幽默、甚至靠下三路，只要有流量，你就是人上人，所以我目前的规划是从游戏主播和自媒体两个方向发展。

    首先来说自媒体吧，有一类博主，他们的视频就是去说，去讲，不涉及视频剪辑，这对于我来讲，正好，因为如果涉及到剪辑或者需要提前准备视频脚本，那耗费的时间太多了，在目前的规划中，确实抽不出时间去做这个，所以我要做的类型就是：举起手机就是说，初步计划是先去说这个 "三年计划"，之后每天就当是去总结一天，看看自己有没有按照每天所说的那样去做了，这样做，既锻炼了自己的表达能力，也算是对自己的一个监督，同时自媒体还可能成为一个产出性爱好。

    其次是游戏主播，一是我确实比较羡慕那些游戏主播的生活，有人喜欢他们，他们也会有很多主播朋友，真的太舒服了，而且这个行业是非常有发展空间的，如果积累了几十个固定观众，那么未来只会越来越多，同时，现在直播的目的也不是火起来，核心与本质是去积累第一批原始观众，同时去学会如何直播有节目效果，因为直播行业在未来也不会落幕，年轻人对于一些主播，会产生依赖的，就像我本人。具体播什么，播多长时间。

    自媒体与直播的具体规划 + 记录在 "近期自媒体与直播计划与记录"。

    技术博客

    技术播客主要产出在自己的微信公众号、个人网站、csdn 等平台。可以是这一天的总结，也可以是一个最小 mvp 的分析与实现。

    理财

    "复利是世界上第八大奇迹"，理财也是每个人都应该学习的，因为钱会越来越不值钱，放到银行里，吃的那点利息，连通货膨胀都跑不赢！

    在理财方面，我的主要规划是，不断地学习股票、基金的相关知识，最终目的是熟悉，了解这个世界的经济体系，为什么它会涨、为什么它会降。学习理财的方式主要有三种，一是阅读经典书籍，比如《穷爸爸，富爸爸》 等等，二是去实操基金，我的手里现在有 10000 元本金用来去投资到基金中，我要真正的感受，是不是如巴菲特所言："通过定投指数基金，一个什么也不懂的业余投资者，往往能够战胜大部分专业基金经理。" 我也要去学习，怎么样去配置自己的基金，怎么去获得稳定的收益。三是去看一些资料，比如比较厉害的博主的视频，博客等，在精华中学习。

    时间就是最大的杠杆。越早学习理财，越好。具体的理财学习规划和记录在 "理财学习规划和记录"

    其他认知

    我现在，首要的目的就是沉淀，从多个维度提升自己，对于结婚，买房，买车，我绝对不考虑，房子、车子不是资产，只有钱才是真正的资产，我要做的就是攒攒攒。
