首先来说**自媒体**吧，有一类博主，他们的视频就是去说，去讲，不涉及视频剪辑，这对于我来讲，正好，因为如果涉及到剪辑或者需要提前准备视频脚本，那耗费的时间太多了，在目前的规划中，确实抽不出时间去做这个，所以我要做的类型就是：举起手机就是说，初步计划是先去说这个 **"三年计划"，之后每天就当是去总结一天，看看自己有没有按照每天所说的那样去做了，这样做，既锻炼了自己的表达能力，也算是对自己的一个监督，同时自媒体还可能成为一个**产出性爱好。

**"复利是世界上第八大奇迹"，**理财也是每个人都应该学习的，因为钱会**越来越不值钱**，放到银行里，吃的那点利息，连**通货膨胀**都跑不赢！

在理财方面，我的主要规划是，不断地学习股票、基金的相关知识，最终目的是熟悉，了解这个世界的经济体系，为什么它会涨、为什么它会降。学习理财的方式主要有三种，**一是**阅读经典书籍，比如《穷爸爸，富爸爸》 等等，**二是**去实操基金，我的手里现在有 10000 元本金用来去投资到基金中，我要真正的感受，是不是如**巴菲特**所言：**"通过定投指数基金，一个什么也不懂的业余投资者，往往能够战胜大部分专业基金经理。"** 我也要去学习，怎么样去配置自己的基金，怎么去获得稳定的收益。**三是**去看一些资料，比如比较厉害的博主的**视频，博客**等，在**精华**中学习。

**时间就是最大的杠杆**。越早学习理财，越好。具体的理财学习规划和记录在 "理财学习规划和记录"
