## NLP 导论

### NLP 概述

自然语言处理（Natural Language Processing, NLP），是人工智能领域的一个重要分支。自然语言，指人类日常使用的语言（如中文、英文），NLP 的目标是让计算机“理解”或“使用”这些语言。

### NLP 常见的任务

自然语言处理包含多个子任务，但是最常见的就是下面这几种：

- 文本分类：对整段文本进行判断或者归类
- 序列标注：给一段文本中的每个词或者字打上特定的标签
- 文本生成：根据已有内容去生成新的自然语言文本
- 信息抽取：从文本中提取出结构化的信息
- 文本转换：将一种文本转换为另一种形式

### NLP 学习环境的准备

建议通过 uv 新建一个专门的 NLP 学习环境，具体操作如下：

``` python
# 1. 创建虚拟环境
uv venv --python=3.12
# 2. 激活
.\.venv\Scripts\activate
# 3. 安装依赖（pytorch 通过 uv 的安装方法在深度学习的笔记中已经给出了，如果嫌麻烦，直接 uv pip 就行）
uv add xxx
```
